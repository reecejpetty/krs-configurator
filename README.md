# KRS Online Configurator

The new version of the KRS Online Configurator built with React. Access app [here](https://reecejpetty.github.io/krs-configurator/).
Project is currently in development. Many features have not been rigorously tested or even implemented. 

## New Features
- Modern and sleek UI re-design
- Mobile interface
- Re-order current sequence items
- Delete any current sequence item
- Edit existing button sequences
- Pin Bumpbar layout to top of screen
- 20/30 button layout toggle
- Helpful tooltips to explain various parts of the app

## In Development
- Improve how upload files handles strings
- Improvements to mobile experience
- Improved documentation and tooltips
- Integration with official KRS Bumpbar iOS app
